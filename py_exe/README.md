# Device Verification Tool – Quick Start Guide
## 1. Using the Blockly Toolbox

Drag blocks from the Toolbox on the left into the workspace to build your test flow.

Connect the blocks freely to design your test program logic.

## 2. Saving the Project

After completing your design, click the Save Workspace button on the right.

Workspace JSON: Saves the current state of the Blockly workspace for future loading.

Output File: The system automatically generates a test output file for hardware execution.

## 3. Running Tests

After saving, use the external SCCB tool.

Sequentially send the generated output file to the board to complete the test process.

## 4. Clear / Reset

Click Clean Workspace to clear the workspace and start a new project.
